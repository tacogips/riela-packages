# riela-workflow-skill-creator-skill

Skill package for authoring packaged skills that teach agents how to use a
Riela workflow package.

This is intentionally separate from `riela-workflow-creator-skill`, which
owns the workflow bundle itself.

The useful artifacts are:

- `skills/agents/AGENTS.md`
- `skills/codex/riela-workflow-use-skill/SKILL.md`
- `skills/claude/riela-workflow-use-skill/SKILL.md`
