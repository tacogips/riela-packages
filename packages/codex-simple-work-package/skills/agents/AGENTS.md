# Project Codex Simple Work Package

This package contains the `codex-simple-work-package` rielflow workflow and
Codex skill.

Use this workflow for small code or documentation changes under a directory when
no dedicated workflow is more appropriate. Preserve the loop contract: the
review step returns to implementation only for `high` or `middle` findings, and
final output occurs only after the latest review accepts the change.

Update `rielflow-package.json` digests after edits.
