# riela-package-installer-skill

Skill-only package for bootstrapping the `riela-package` package-management
skill from the `tacogips/riela-packages` registry.

The useful artifacts are:

- `skills/codex/riela-package-installer/SKILL.md`
- `skills/claude/riela-package-installer/SKILL.md`

The workflow bundle is a minimal install marker required by the current
riela package manifest format.
