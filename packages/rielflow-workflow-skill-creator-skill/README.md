# rielflow-workflow-skill-creator-skill

Skill package for authoring packaged skills that teach agents how to use a
Rielflow workflow package.

This is intentionally separate from `rielflow-workflow-creator-skill`, which
owns the workflow bundle itself.

The useful artifacts are:

- `skills/agents/AGENTS.md`
- `skills/codex/rielflow-workflow-use-skill/SKILL.md`
- `skills/claude/rielflow-workflow-use-skill/SKILL.md`
